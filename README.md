## 🚀 快速开始

### 手机端使用

直接打开网页即可使用，无需安装任何 App：

**访问地址**：[https://p2pfile.cool-emoji.com/](https://p2pfile.cool-emoji.com/)

![手机端界面](CF-Workers/img/mobile.png)

#### 发送文件

1. 打开网页，点击「生成」按钮获取传输码
2. 点击「连接」，等待对方接入
3. 连接成功后，点击「添加文件」选择要发送的文件
4. 点击「发送全部」，等待对方确认接收

#### 接收文件

1. 输入对方显示的传输码
2. 点击「连接」，等待发送方发送文件
3. 收到文件请求后，点击「接收」或「拒绝」
4. 接收完成后，文件自动保存到「下载」文件夹

---

### 电脑端使用

下载安装即可使用，无需注册账号：

**下载地址**：[闪联灵狐.exe](https://github.com/lty0311/file-p2p/releases)

![电脑端界面](CF-Workers/img/pc.png)

#### 发送文件

1. 运行程序，点击「生成」获取传输码
2. 点击「连接」，等待对方
3. 点击「选择要发送的文件」，选择文件
4. 点击「确认发送」，等待对方确认

#### 接收文件

1. 输入发送方的传输码
2. 点击「连接」
3. 收到文件请求时，点击「接收」选择保存位置，或「拒绝」

## 🔧 技术原理

本项目采用 **WebRTC** 技术实现真正的 P2P（点对点）传输：

```
┌─────────────┐                    ┌─────────────┐
│   发送方    │                    │   接收方    │
│  (手机/PC)  │                    │  (手机/PC)  │
└──────┬──────┘                    └──────┬──────┘
       │                                  │
       │  1. 输入相同传输码                │
       │  2. 连接信令服务器（仅交换地址）   │
       │                                  │
       │     ┌──────────────┐             │
       │────▶│  信令服务器  │◀────────────│
       │     │ (仅转发SDP) │             │
       │     └──────────────┘             │
       │                                  │
       │  3. P2P 直连建立                 │
       │◀────────────────────────────────▶│
       │                                  │
       │  4. 文件数据直传                  │
       │◀────────────────────────────────▶│
       │     (不经过服务器)               │
       │                                  │
```

**关键特点**：
- 信令服务器仅用于交换网络地址（SDP/ICE），**不接触文件数据**
- 文件通过 WebRTC DataChannel 直连传输
- 支持 NAT 穿透，无需公网 IP
- 分块传输（256KB/块），支持大文件和进度显示
